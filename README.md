# 🎧 Pink Music UI

A modern **music player UI concept** built using only **HTML and CSS**.  
No JavaScript. No frameworks. Pure interface design.

---

## ⚡ Features

- Dark premium UI design system  
- Pink gradient theme with neon highlights  
- Smooth hover interactions (buttons + cards)  
- Active state effects for player controls  
- Playlist section with modern card layout  
- Bottom navigation UI  
- Mobile-first centered layout  

---

## 🧩 UI Components

- Now Playing Section  
- Album Artwork Card  
- Playback Controls (Play / Next / Prev)  
- Progress Bar UI  
- Playlist Cards  
- Bottom Navigation Bar  

---

## 🖼️ Design Style

- Minimal + premium aesthetic  
- Soft shadows with depth layering  
- Neon pink accent system  
- Glass-like dark surfaces  
- Inspired by Spotify / Apple Music UI concepts  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Flexbox, Gradients, Hover States)  
- Google Fonts (Inter)  

---

## ⛔ Project Type

- Not a functional music player  
- No JavaScript used  
- No audio logic or backend  
- Pure UI/UX design concept
